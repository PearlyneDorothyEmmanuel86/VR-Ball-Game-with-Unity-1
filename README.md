# VR-Ball-Game-with-Unity-1

Download to play the game here: https://drive.google.com/file/d/1byDuijBsP17fsUAOUbmP3osMUMGuR486/view?usp=drive_link

Description of game: There are 2 red x ray interactor components, one for the righthand controller and one for the lefthand controller. The controllers can teleport once in the pink teleportation area (by holding T and pressing L). Once the lefthand controller is in the teleporting area a grey cylinder will be attached to it. When the righthand controller is in the teleporting area the grey cylinder will be attached to it instead. If both the righthand and lefthand controllers are in the teleporting area, the righthand controller will have the grey cylinder attached to it.
